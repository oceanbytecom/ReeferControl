Features

Everything in one app

From temperature logging to spare parts inventory — built for the real workflow on deck.

🌡

Temperature monitoring

Log supply and return temps per container. Spot over-set-point deviations instantly with colour-coded status.

🚨

Alarm management

File alarms with one tap. Timestamps, container ID, and deviation data captured automatically.

📦

Spare parts stock

Search parts by number or description, track quantities, scan barcodes, and log every movement with history.

📋

Orders & reports

Generate and export shift reports and part orders. Import containers and parts from Excel in seconds.

🗺

Ship route tracking

Set your voyage route with ETA/ETD per port. See which port is next and how much time remains.

📡

Database sharing via QR

Share the full database between devices by scanning a QR code. No cloud, no internet, no accounts.

📥

Remote manual updates

Manuals and alarm references update automatically from the internet when new versions are published.

🏆

Achievement system

50 achievements that reward thorough inspections, accurate temperature data, and zero missed containers.

Installation

Three steps to get started

No Google Play, no account, no subscription.

1

Download the APK file

Tap the Download button — your browser saves reefer-control.apk to your Downloads folder. The file is about 18 MB.

✓ No Google account needed

2

Allow installation from unknown sources

Go to Settings → Security → Install unknown apps and enable it for your browser. You only do this once.

✓ Works on Android 8 and newer

3

Open the file and tap Install

Open reefer-control.apk from your notifications or Files app. Tap Install and the app appears on your home screen.

✓ Done in under 30 seconds
